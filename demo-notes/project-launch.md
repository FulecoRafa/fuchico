# Project Launch

Open this folder ("Open Folder" → `demo-notes`) and click the calendar
icon in the activity bar to see these tasks, todos, and events show up
in the global Tasks & Calendar view.

## Syntax

- Task: `- [ ] text` or `- [x] text`, optionally with a due date via
  `📅 YYYY-MM-DD` (and an optional `HH:MM`) anywhere in the text.
- Todo: a bare `TODO: text` line — listed, but not toggleable like a task.
- Event: a line that *starts* with `📅 YYYY-MM-DD [HH:MM] text` and has no
  checkbox — calendar-only, not actionable.
